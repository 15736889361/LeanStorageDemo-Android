一个 LeanCloud 综合型的 Demo，演示了大多数功能:

* 对象的增删改查
* 用户系统
* 文件上传下载
* 子类化

需要您通过查看代码来学习。


## 如何运行

* 用 Android Studio 打开本项目

## 登录后台查看数据

您可以在`com.example.avoscloud_demo.DemoGroupActivity`修改成您自己的应用 id 和 key。强烈建议这样，一边查看自己的后台，一边运行 Demo。当在 Demo 运行代码来增删改查时，就可以在自己的后台看到相应的数据变化。
